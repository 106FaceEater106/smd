## Usage

### First you need to install all dependencies
```
pip3 install -r requirements.txt
```

### Make file executable
```
chmod +x main.py
./main.py
```

### Example of Spotify URI Code
```
spotify:track:7lEptt4wbM0yJTvSG5EBof
```
